# Image slots

Drop these files into this `images/` folder, then in `index.html`:
1. Delete the `<div class="image-slot" ...>` placeholder line right above each comment.
2. Uncomment the matching `<!-- <img ...> -->` line just below it.

| File to add             | Used for                        | Suggested size      |
|--------------------------|----------------------------------|----------------------|
| `profile.jpg`             | About section headshot           | Square, 600×600+     |
| `project-loan.jpg`        | Loan Prediction System card      | 800×500 (16:10)      |
| `project-jarvis.jpg`      | AI Virtual Assistant card        | 800×500 (16:10)      |
| `project-placement.jpg`   | College Placement Portal card    | 800×500 (16:10)      |
| `project-recipe.jpg`      | Recipe Generator card            | 800×500 (16:10)      |
| `project-fakereview.jpg`  | Fake Review Detection card       | 800×500 (16:10)      |

Any format works (jpg/png/webp) — just update the `src` extension to match.
